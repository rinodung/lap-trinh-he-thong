This directory contains Perl programs to generated opcode tables.

The tables can be in .tex format, or as plain text files.

The opcodes area generated by assembling and disassembling programs

FILES:

	genlea.pl

Generates opcodes for instructions of the form leaq (r1,r2,1), %rax

	genmov.pl

Generates opcodes for movb, movw, movl and movq instructions

	gennop.pl	

Generates opcodes for limited class of functions nops

	genpop.pl	

Generates opcodes for popq instructions
